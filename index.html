About

With a background in sales and big data analysis, I've always been passionate about communicating information through good design, whether by making sales material to present to clients or creating data visualization dashboards for clients. After having worked at an e-commerce tech giant in Japan for 3 years and at a market research tech company in Silicon Valley, I took a leap of faith into a career in web development. I started off making simple websites on my own, and from there started building full-stack applications through the web development immersive program at General Assembly.

In every web development project, I spend a good amount of time writing user stories, sketching the wireframe, and modeling the entities of my app. I believe that, without thoughtful planning initially, you cannot build an app that users will find useful.

Projects

Kioku

Full stack flashcard app

A flashcard single page application that allows users to create flashcards to study Japanese words and short phrases. Supports user authentication and CRUD actions (create, read, edit, and delete flashcards).

The back-end API is built with Ruby on Rails using a PostgreSQL database, and implements RESTful routes. The front-end was built with JavaScript, jQuery, AJAX, Handlebars, Sass, and Bootstrap. I also implemented use of a Text-to-Speech API to add audio of the Japanese pronunciation to each word.

Live GitHub

State Trotter

Full stack bucket list app

A bucket list single page application for making goals for each of the 50 US states, built collaboratively in a team of four developers. Supports user authentication and CRUD actions (create, read, edit, and delete bucket lists items). The US map is built with JQVMap, a jQuery plugin, that renders Vector Maps.

The back-end API is built with NodeJS and Express, using a MongoDB database, and implements RESTful routes. The front-end was built with JavaScript, jQuery, AJAX, Handlebars, Sass, and Bootstrap.

Live GitHub

TicTacToe

Front-end Tic tac toe game

My first front-end single page application for a seemingly simple game of Tic-Tac-Toe. The game logic is built with JavaScript. The game sends AJAX requests to a Rails API for user authentication and storing game statistics. Users can create new tic-tac-toe games and view past game statistics.

The front-end is built with HTML, CSS, JavaScript, jQuery, and AJAX.
